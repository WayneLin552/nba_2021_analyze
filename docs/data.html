We have to admit that because those data is collected by NBA official with most advanced equipment, so it is quite clean and well structured. By carefully checking every rows and columns, we finally find some uncleaned data: a missing column name and not clear percentage numbers.

For further transformation, we can consider using standard scaler or log scaler for those numerical features, but this final project so far only focus on data visualization and analyzation, not about training a machine learning model, so we do not have to further scale those data. For data cleaning and transformation step, it is enough so far.
